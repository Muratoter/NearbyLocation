# NearbyLocation
Google Nearby Api kullanarak seçilen kategorideki yakın konumların detayların gösterilmesi

Seçilen kategorilere göre 1000 metreye kadar yakın olan kayıtlı yer bilgilerini getiren Android uygulamasıdır.

<b>Kategoriler;</b>
<li>Hastaneler</li>
<li>Marketlar</li>
<li>Restoranlar</li>
<li>Okullar</li>
